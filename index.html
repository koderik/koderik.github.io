KTH Tentor Twitter Bot

  • KTH Exam Twitter Bot is a utility developed to facilitate regular updates on KTH exams by automatically tweeting the exam schedules on a daily basis. This project is built using Python, leveraging libraries such as Pandas and Tweepy for data manipulation and Twitter API interaction respectively. Notably, the bot efficiently scrapes and reformats KTH's exam calendar data, making it a valuable tool for students and educators seeking up-to-date exam information.
